Game ID format
S(06 to 12 for number of roles)(0 or 1 for sheriff mode)(A or E for gamemode: kill_all / kill_either)(Multiple Role IDs of minimum 0 and maximum 12)L
Each role id consists of one character followed by 4 digits:
- The first character represents category.
- The first two digits represents role ID.
- The last two digits represents number of the role (from 01 to 12).
Minimum length of Game ID: 6
Maximum length of Game ID: 66


Role ID

Werewolves (W)
00 - Werewolf

Villagers (V)
00 - Villager

God Identities (G)
00 - Seer
01 - Hunter
02 - Witch (not added yet)
