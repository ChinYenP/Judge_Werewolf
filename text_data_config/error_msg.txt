Display Error Message SOP:

For user side error, use normal display.

For server side error:
1. console.error the following:
- which part of the file goes wrong
- which part of the file goes wrong (index if needed)
- a description if needed.
2. At user side, display normally but use code to specify the type of error only.
- DO NOT display the location of the error.



Error Codes:

D: Database Error
D1: Insert data into database
D2: Delete data from database
D3: Modify data in database
D4: Search data in database
D5: Wrong value of data

C: Command Error
C1: Undefined prefix
C2: Error during executing command
C3: Unexpected prefix error
C4: Command not found or validated

S: Select Menu Error

DSPY: Display Error (usually occurs in get_display.js)

M: Message Error
M1: Message edit error

U: Undefined Error
